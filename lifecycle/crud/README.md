CRUD
===

To run the app, in the vs code open terminal and terminal run:

`cd backend` <br>
`npm i` <br>
`npm start` <br>

To run the app, in the vs code open another terminal and run:

`cd frontend` <br>
`npm i` <br>
`npm start` <br>
